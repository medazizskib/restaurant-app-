# Restaurant App

This is an interactive restaurant web app built using **HTML, CSS, and JavaScript**.

## Features
- Display a menu of dishes
- Add items to the cart
- View total price
- Checkout with confirmation

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a web browser.

## Usage
- Browse the menu.
- Click "Add to Cart" to add items.
- View your cart and total price.
- Click "Checkout" to confirm your order.
